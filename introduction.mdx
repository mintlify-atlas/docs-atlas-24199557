# Stripe UI Kit

Easily display your customers' Stripe information in your own app. Works with any authentication provider.

The Stripe UI Kit is a collection of React components built for Next.js that lets you quickly add Stripe customer data views to your application. Display customer details, invoices, payments, and subscriptions with minimal setup.

## Key features

- **View customer information** - Display and edit customer details including name, email, and address
- **Built with Next.js and TanStack Query** - Server-side data fetching with automatic caching and revalidation
- **Styled with ShadCN UI components** - Beautiful, accessible components that you can customize
- **Fully customizable and extensible** - Copy the source code and modify it to fit your needs
- **Uses native Stripe API calls** - Direct integration with the Stripe API for real-time data
- **Authentication provider agnostic** - Works with Clerk, NextAuth, or any auth solution

## What's included

The kit includes four main components:

- `CustomerCard` - View and edit customer information
- `CustomerInvoices` - Display all customer invoices in a table
- `CustomerPayments` - Show payment history with card details
- `CustomerSubscriptions` - List active and past subscriptions

## Requirements

Before you begin, make sure you have:

- A Next.js project (App Router)
- A Stripe account with API keys
- An authentication system (Clerk, NextAuth, etc.)

<Note>
  This kit uses the Next.js App Router and requires Next.js 15 or later.
</Note>
